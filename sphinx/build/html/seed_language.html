SeedLanguage¶

Different seed phrases support different languages, yet. To handle this differences, the ots.seed_language.SeedLanguage class provides a way to manage the language of seed phrases. ots.seed_language.SeedLanguage “knows” which languages are supported by each ots.enums.SeedType.

classmethod setDefaultLanguage(seedType: SeedType, language: SeedLanguage) → None¶

Sets the default SeedLanguage instance for the given seed type, so you can manage the default language for seed phrases. By default there are no defaults set, so you must call this method to set a default language for each SeedType you want to use.

Parameters:

  • seedType (SeedType) – The type of seed to set the default language for.

  • language (SeedLanguage) – The SeedLanguage instance to set as default.
