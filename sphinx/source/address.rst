Address
=======

Any :py:class:`ots.address.Address` object represents a valid Monero address. To handle
unknown addresses :py:class:`ots.address.AddressString` can be used.

.. seealso::

   Monero Docs

   `Address types <https://docs.getmonero.org/public-address/>`_
   about Standard address, Subaddress, and Integrated addresses.
