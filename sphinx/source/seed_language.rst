SeedLanguage
============

Different seed phrases support different languages, yet. To handle this differences,
the :py:class:`ots.seed_language.SeedLanguage` class provides a way to manage the language of seed phrases. :py:class:`ots.seed_language.SeedLanguage` "knows" which languages are supported by each :py:class:`ots.enums.SeedType`.
